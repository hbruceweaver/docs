# Higgsfield API

Build AI-powered video and image generation into your applications with a simple, unified API.

## What is Higgsfield API?

Higgsfield API provides managed infrastructure for AI video and image generation. Instead of managing API keys, rate limits, and job queues yourself, we handle:

- **Job Queue Management** - Automatic queuing, retries, and status tracking
- **Account Pooling** - Distribute load across multiple generation accounts
- **Webhook Callbacks** - Get notified when jobs complete or fail
- **Usage Tracking** - Monitor credits and usage per organization

## Quick Example

Generate an image with Nano Banana 2:

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "nano_banana_2",
    "params": {
      "prompt": "a serene mountain landscape at sunset",
      "width": 1024,
      "height": 1024,
      "batch_size": 1,
      "aspect_ratio": "1:1",
      "resolution": "1k"
    }
  }'
```

**Response:**

```json
{
  "id": "j57abc123def456"
}
```

Poll for completion:

```bash
curl https://higgsfield.ai/api/jobs?id=j57abc123def456 \
  -H "Authorization: Bearer hf_your_api_key"
```

**Response (completed):**

```json
{
  "id": "j57abc123def456",
  "type": "nano_banana_2",
  "status": "completed",
  "results": {
    "raw": {
      "type": "image",
      "url": "https://cdn.higgsfield.ai/..."
    }
  }
}
```
