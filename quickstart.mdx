## 1. Get Your API Key

1. Sign in to the [Higgsfield Dashboard](https://higgsfield.dev/dashboard)
2. Navigate to **API Keys**
3. Click **Create API Key**
4. Copy your key (starts with `hf_`)

<Warning>
Store your API key securely. It cannot be retrieved after creation.
</Warning>

## 2. Generate Your First Image

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "nano_banana_2",
    "params": {
      "prompt": "A serene mountain landscape at golden hour, dramatic clouds, photorealistic",
      "width": 1024,
      "height": 1024,
      "batch_size": 1,
      "aspect_ratio": "1:1",
      "resolution": "2k"
    }
  }'
```

**Response:**

```json
{
  "id": "j57abc123def456"
}
```

## 3. Poll for Completion

Jobs take 10-60 seconds. Poll the status:

```bash
curl "https://higgsfield.ai/api/jobs?id=j57abc123def456" \
  -H "Authorization: Bearer hf_your_api_key"
```

**Response (completed):**

```json
{
  "id": "j57abc123def456",
  "type": "nano_banana_2",
  "status": "completed",
  "results": {
    "raw": {
      "type": "image",
      "url": "https://cdn.higgsfield.ai/image.png"
    }
  }
}
```

## 4. Download Your Image

The `results.raw.url` is a direct download link:

```bash
curl -o image.png "https://cdn.higgsfield.ai/image.png"
```

## Job Status Flow

```
queued → in_progress → completed
                    ↘ failed
```

| Status | Description |
|--------|-------------|
| `queued` | Waiting for available account |
| `in_progress` | Actively processing |
| `completed` | Success, results available |
| `failed` | Failed, check `failReason` |

## Using External IDs

Track jobs with your own identifiers:

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "nano_banana_2",
    "externalId": "order-12345",
    "params": { ... }
  }'
```

Query by external ID:

```bash
curl "https://higgsfield.ai/api/jobs?externalId=order-12345" \
  -H "Authorization: Bearer hf_your_api_key"
```
