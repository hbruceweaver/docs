## Overview

Nano Banana 2 is a fast, general-purpose text-to-image model. Supports multiple aspect ratios and resolutions up to 2K.

**Best for:** Rapid iteration, testing prompts, general-purpose image generation.

## Quick Start

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "nano_banana_2",
    "params": {
      "prompt": "A serene mountain landscape at golden hour, dramatic lighting, photorealistic",
      "width": 1024,
      "height": 1024,
      "batch_size": 1,
      "aspect_ratio": "1:1",
      "resolution": "1k"
    }
  }'
```

## Parameters

<ParamField body="prompt" type="string" required>
  Text description of the image to generate.
</ParamField>

<ParamField body="width" type="number" required>
  Output width in pixels. See dimensions table.
</ParamField>

<ParamField body="height" type="number" required>
  Output height in pixels. See dimensions table.
</ParamField>

<ParamField body="batch_size" type="number" default="1">
  Number of images to generate (1-4).
</ParamField>

<ParamField body="aspect_ratio" type="string" required>
  Image aspect ratio. See supported ratios below.
</ParamField>

<ParamField body="resolution" type="string" required>
  Output resolution: `1k` or `2k`.
</ParamField>

<ParamField body="input_images" type="array">
  Optional reference images for image-to-image generation.
</ParamField>

<ParamField body="is_storyboard" type="boolean" default="false">
  Enable storyboard mode.
</ParamField>

<ParamField body="is_zoom_control" type="boolean" default="false">
  Enable zoom control.
</ParamField>

## Aspect Ratios

| Ratio | Description |
|-------|-------------|
| `1:1` | Square |
| `16:9` | Landscape (widescreen) |
| `9:16` | Portrait (mobile) |
| `4:3` | Standard landscape |
| `3:4` | Standard portrait |
| `3:2` | Photography landscape |
| `2:3` | Photography portrait |
| `21:9` | Ultra-wide |
| `5:4` | Medium format |
| `4:5` | Instagram portrait |

## Dimensions by Resolution

### 1K Resolution

| Aspect Ratio | Width | Height |
|--------------|-------|--------|
| 1:1 | 1024 | 1024 |
| 16:9 | 1344 | 768 |
| 9:16 | 768 | 1344 |
| 4:3 | 1184 | 864 |
| 3:4 | 864 | 1184 |
| 3:2 | 1248 | 832 |
| 2:3 | 832 | 1248 |
| 21:9 | 1536 | 672 |
| 5:4 | 1152 | 896 |
| 4:5 | 896 | 1152 |

### 2K Resolution

Dimensions are the same base values; the `2k` resolution parameter enables internal upscaling for higher quality output.

## Tips

1. **Be descriptive** - Include style, lighting, composition details
2. **Use quality modifiers** - "photorealistic", "4K", "detailed", "professional"
3. **Specify lighting** - "golden hour", "soft diffused light", "dramatic shadows"
4. **Include composition** - "close-up", "wide shot", "aerial view"
5. **Batch for variety** - Use `batch_size: 4` to get variations, then pick the best

## Batch Results

When `batch_size` > 1, results contain multiple images. Poll the job status and you'll receive URLs for all generated images in the results.
