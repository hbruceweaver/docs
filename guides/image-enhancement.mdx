## Overview

Post-process generated images to improve quality:

- **Skin Enhancer** - Refine skin texture in portraits
- **Topaz Upscale** - Increase resolution up to 4x

## Enhancement Pipeline

```
Generate Image → Skin Enhancer (if portrait) → Topaz Upscale
```

## Skin Enhancer

Improves skin detail in portraits. Apply after generating headshots or fashion imagery.

### When to Use

- AI-generated portraits with skin artifacts
- Headshots that need refinement
- Beauty/fashion photography
- Any image with visible faces

### Example

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "skin_enhancer",
    "params": {
      "input_image": {
        "id": "portrait-123",
        "type": "text2image_soul_job",
        "url": "https://cdn.higgsfield.ai/portrait.png"
      },
      "width": 1536,
      "height": 2048,
      "output_width": 1536,
      "output_height": 2048,
      "preset_id": "default"
    }
  }'
```

### Input Types

| Type | Source |
|------|--------|
| `text2image_soul_job` | Soul model output |
| `nano_banana_2_job` | Nano Banana 2 output |
| `seedream_v4_5_job` | Seedream output |
| `media_input` | Uploaded image |

### Cost

1 credit per enhancement.

## Topaz Upscale

Increase image resolution using AI upscaling.

### When to Use

- Preparing images for print
- Large format displays
- Increasing low-res image quality
- Final production assets

### Example

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "topaz_image",
    "params": {
      "input_image": {
        "id": "enhanced-portrait",
        "type": "skin_enhancer_job",
        "url": "https://cdn.higgsfield.ai/enhanced.png"
      },
      "width": 1536,
      "height": 2048,
      "output_width": 6144,
      "output_height": 8192,
      "denoise": 0.2,
      "sharpen": 0.3,
      "face_enhancement": true,
      "face_enhancement_creativity": 0.3,
      "face_enhancement_strength": 0.5,
      "model": "Standard V2"
    }
  }'
```

### Scale Options

| Scale | Example | Use Case |
|-------|---------|----------|
| 2x | 1024 → 2048 | Web, general |
| 3x | 1024 → 3072 | Large displays |
| 4x | 1024 → 4096 | Print, max quality |

### Parameters

| Parameter | Range | Notes |
|-----------|-------|-------|
| `denoise` | 0-1 | Remove noise (0.1-0.3 typical) |
| `sharpen` | 0-1 | Edge sharpening (0.2-0.4 typical) |
| `face_enhancement` | bool | Enable for portraits |
| `face_enhancement_creativity` | 0-1 | How much AI can interpret |
| `face_enhancement_strength` | 0-1 | Strength of enhancement |

### Cost

1 credit per upscale (flat rate regardless of scale).

## Full Pipeline Example

### Step 1: Generate Portrait

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "text2image_soul",
    "params": {
      "prompt": "Professional headshot, natural lighting, neutral background",
      "quality": "1080p",
      "aspect_ratio": "3:4",
      "style_id": "portrait-style-uuid",
      "style_strength": 0.7,
      "seed": 42,
      "batch_size": 1
    }
  }'
```

Cost: 2 credits

### Step 2: Wait and Get URL

```bash
curl "https://higgsfield.ai/api/jobs?id=SOUL_JOB_ID" \
  -H "Authorization: Bearer hf_your_api_key"
```

### Step 3: Enhance Skin

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "skin_enhancer",
    "params": {
      "input_image": {
        "id": "soul-portrait",
        "type": "text2image_soul_job",
        "url": "URL_FROM_STEP_2"
      },
      "width": 1536,
      "height": 2048,
      "output_width": 1536,
      "output_height": 2048,
      "preset_id": "default"
    }
  }'
```

Cost: 1 credit

### Step 4: Upscale

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "topaz_image",
    "params": {
      "input_image": {
        "id": "enhanced",
        "type": "skin_enhancer_job",
        "url": "URL_FROM_STEP_3"
      },
      "width": 1536,
      "height": 2048,
      "output_width": 4608,
      "output_height": 6144,
      "denoise": 0.2,
      "sharpen": 0.25,
      "face_enhancement": true,
      "face_enhancement_creativity": 0.3,
      "face_enhancement_strength": 0.5,
      "model": "Standard V2"
    }
  }'
```

Cost: 1 credit

**Total Pipeline Cost: 4 credits**

Result: High-resolution, skin-enhanced portrait (4608x6144)

## Recommendations by Content Type

| Content Type | Skin Enhancer | Topaz Settings |
|--------------|---------------|----------------|
| Headshots | Yes | face_enhancement: true, strength: 0.5 |
| Landscapes | No | denoise: 0.1, sharpen: 0.35 |
| Products | No | denoise: 0.15, sharpen: 0.4 |
| Fashion | Yes | face_enhancement: true, strength: 0.3 |
| Abstract | No | denoise: 0.1, sharpen: 0.2 |

## Tips

1. **Enhance before upscale** - Skin enhancement at native resolution, then upscale
2. **Don't over-process** - Subtle settings usually look better
3. **Check quality** - Compare before/after to verify improvement
4. **Match scale to use** - 2x for web, 4x for print
5. **Skip when unnecessary** - Not every image needs enhancement

## When to Skip Enhancement

| Scenario | Skip |
|----------|------|
| Landscape photos | Skin Enhancer |
| Already high-res | Topaz |
| Abstract/artistic | Both (often) |
| Quick iteration | Both |
| Web-only use | Often skip 4x upscale |

## Automation Script

JavaScript example for full pipeline:

```javascript
async function enhancePortrait(apiKey, sourceUrl) {
  // Step 1: Skin enhance
  const skinJob = await createJob(apiKey, {
    type: 'skin_enhancer',
    params: {
      input_image: {
        id: 'source',
        type: 'media_input',
        url: sourceUrl
      },
      width: 1024,
      height: 1365,
      output_width: 1024,
      output_height: 1365,
      preset_id: 'default'
    }
  });

  const enhancedUrl = await waitForJob(apiKey, skinJob.id);

  // Step 2: Upscale
  const upscaleJob = await createJob(apiKey, {
    type: 'topaz_image',
    params: {
      input_image: {
        id: 'enhanced',
        type: 'skin_enhancer_job',
        url: enhancedUrl
      },
      width: 1024,
      height: 1365,
      output_width: 4096,
      output_height: 5460,
      denoise: 0.2,
      sharpen: 0.3,
      face_enhancement: true,
      face_enhancement_creativity: 0.3,
      face_enhancement_strength: 0.5,
      model: 'Standard V2'
    }
  });

  return await waitForJob(apiKey, upscaleJob.id);
}
```
