## Overview

Create 5-10 second videos from text descriptions using the Kling 2.6 model. Supports optional AI-generated audio.

## Prerequisites

- API key ([get one here](https://higgsfield.dev/dashboard/api-keys))
- Credits balance (10 credits per 5s video)

## Step 1: Create the Job

```bash
curl -X POST https://higgsfield.ai/api/jobs \
  -H "Authorization: Bearer hf_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "kling_26",
    "params": {
      "prompt": "A woman walking on a beach at sunset, golden hour lighting, waves gently lapping",
      "width": 1920,
      "height": 1080,
      "duration": 5,
      "aspect_ratio": "16:9",
      "sound": true,
      "cfg_scale": 0.5,
      "enhance_prompt": true
    }
  }'
```

**Response:**

```json
{
  "id": "j57abc123def456"
}
```

## Step 2: Poll for Completion

Video generation takes 1-3 minutes. Poll the status:

```bash
curl "https://higgsfield.ai/api/jobs?id=j57abc123def456" \
  -H "Authorization: Bearer hf_your_api_key"
```

**Response (in progress):**

```json
{
  "id": "j57abc123def456",
  "type": "kling_26",
  "status": "in_progress"
}
```

**Response (completed):**

```json
{
  "id": "j57abc123def456",
  "type": "kling_26",
  "status": "completed",
  "results": {
    "raw": {
      "type": "video",
      "url": "https://cdn.higgsfield.ai/video.mp4",
      "thumbnail_url": "https://cdn.higgsfield.ai/thumb.webp"
    }
  }
}
```

## Step 3: Download the Video

The `results.raw.url` contains the direct download link. Videos are MP4 format.

```bash
curl -o video.mp4 "https://cdn.higgsfield.ai/video.mp4"
```

## Polling Helper

Here's a JavaScript function to poll until completion:

```javascript
async function waitForVideo(jobId, apiKey) {
  const maxWait = 5 * 60 * 1000; // 5 minutes
  const startTime = Date.now();
  let interval = 5000;

  while (Date.now() - startTime < maxWait) {
    const response = await fetch(
      `https://higgsfield.ai/api/jobs?id=${jobId}`,
      { headers: { Authorization: `Bearer ${apiKey}` } }
    );
    const job = await response.json();

    if (job.status === 'completed') {
      return job.results.raw.url;
    }

    if (job.status === 'failed') {
      throw new Error(`Video generation failed: ${job.failReason}`);
    }

    await new Promise(r => setTimeout(r, interval));
    interval = Math.min(interval * 1.5, 30000);
  }

  throw new Error('Timeout waiting for video');
}
```

## Prompt Tips

### Good Prompts

- "Cinematic drone shot of a forest, morning mist, sun rays through trees"
- "Close-up of coffee being poured, steam rising, warm indoor lighting"
- "Person walking through city streets, neon signs reflecting on wet pavement, night"

### Things to Include

- **Subject** - What/who is in the video
- **Action** - What's happening
- **Setting** - Where it takes place
- **Lighting** - Time of day, light quality
- **Camera** - Movement, angle (optional)

### Things to Avoid

- Vague descriptions ("something cool")
- Too many subjects or actions
- Impossible physics (unless intended)

## With Audio

Set `sound: true` to generate ambient audio that matches the video content:

```json
{
  "type": "kling_26",
  "params": {
    "prompt": "Ocean waves crashing on rocky shore, seagulls flying",
    "width": 1920,
    "height": 1080,
    "duration": 5,
    "aspect_ratio": "16:9",
    "sound": true
  }
}
```

<Note>
Audio generation adds processing time but creates more immersive content.
</Note>

## Duration Options

| Duration | Credits | Notes |

| 5 seconds | 10 | Works in all modes |
| 10 seconds | 20 | Not available in unlimited mode |

## Common Issues

| Issue | Solution |
|-------|----------|
| Job stuck in queued | All accounts busy; wait for availability |
| Blurry output | Be more specific in prompt; try higher cfg_scale |
| Wrong subject | Describe subject clearly at the start of prompt |
| No motion | Include action verbs: "walking", "flowing", "moving" |
