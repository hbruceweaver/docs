---
title: 'List Accounts'
api: 'GET /api/accounts'
description: 'List all generation accounts'
---

## Query Parameters

<ParamField query="includeDisabled" type="boolean" default="false">
  Include disabled accounts in the response
</ParamField>

## Response

<ResponseField name="accounts" type="array">
  Array of account objects
</ResponseField>

## Example

```bash
curl "https://higgsfield.ai/api/accounts" \
  -H "Authorization: Bearer hf_your_api_key"
```

### Include Disabled

```bash
curl "https://higgsfield.ai/api/accounts?includeDisabled=true" \
  -H "Authorization: Bearer hf_your_api_key"
```
